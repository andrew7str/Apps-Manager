# 🇬🇧 English Version

## 🚀 Overview
**Apps Manager Pro** is a high-performance *system orchestration toolkit* for Linux. Whether you're on a server (CLI) or a desktop (GUI), it empowers you to manage applications, drivers, and system configurations with unmatched efficiency.

### 💎 Why Apps Manager Pro?
- **Cross-Distro Intelligence**: Automatically detects `apt`, `dnf`, `pacman`, or `zypper`.
- **Hybrid Interface**: Switch between a powerful Interactive CLI and a stunning "Dark Tron Green" GUI.
- **Universal Support**: Handles native packages, Flatpaks, Snaps, and even direct URL downloads.

---

## 🌟 What's New in v1.1.0?
- **🖥️ Virtualization Hub**: Deploy **VirtualBox** (with auto-license agreement), **GNOME Boxes**, and **Virt-Manager** in one click.
- **🗑️ Pure Control (Uninstall)**: Not just for installing! Now you can **Uninstall** applications directly from the interface.
- **🛠️ Robust Detection**: Smarter package state detection across various Linux flavors.

## 📦 Getting Started

### 1. Clone & Prep
```bash
git clone https://github.com/andrew7str/Apps-Manager.git
cd Apps-Manager
```

### 2. Launch CLI Mode (Server/Power User)
```bash
chmod +x AppsManagerPro.sh
./AppsManagerPro.sh
```

### 3. Launch GUI Mode (Desktop/Premium Experience)
*Requires `customtkinter` (`pip install customtkinter`)*
```bash
python3 AppsManagerProGUI.py
```
